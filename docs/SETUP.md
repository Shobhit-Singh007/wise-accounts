> Step-by-step instructions to set up the development environment.

## 1. Clone Repository

```bash
git clone https://github.com/your-org/wise-accounts.git
cd wise-accounts
```

---

## 2. Backend Setup

### 2.1 Install Dependencies

```bash
cd backend
npm install
```

### 2.2 Database Setup

**Option A: Local PostgreSQL**

```bash
# Create database
psql -U postgres -c "CREATE DATABASE wise_accounts;"

# Or using createdb
createdb -U postgres wise_accounts
```

**Option B: Docker PostgreSQL**

```bash
docker run -d \
  --name wise-accounts-db \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_PASSWORD=postgres \
  -e POSTGRES_DB=wise_accounts \
  -p 5432:5432 \
  postgres:15-alpine
```

### 2.3 Configure Environment

The `.env` file is already configured for local development:

```env
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/wise_accounts?schema=public"
JWT_SECRET="your-super-secret-jwt-key-change-in-production"
JWT_EXPIRY="15m"
PORT=3000
CORS_ORIGIN="*"
```

> Update `JWT_SECRET` to a strong random value for your local setup.

### 2.4 Run Migrations

```bash
npx prisma migrate dev
```

This will:
1. Create all tables in the database
2. Generate the Prisma client
3. Create a migration file

### 2.5 Seed Database (Optional)

```bash
# If a seed file exists:
npx prisma db seed
```

### 2.6 Start Development Server

```bash
npm run start:dev
```

The server starts at:
- API: `http://localhost:3000/api/v1`
- Swagger Docs: `http://localhost:3000/api/docs`

### 2.7 Verify

```bash
# Test register
curl -X POST http://localhost:3000/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{"phone":"+919876543210","name":"Test User","password":"Password@123"}'

# Test login
curl -X POST http://localhost:3000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"+919876543210","password":"Password@123"}'
```

## 3. Android Setup

### 3.1 Open in Android Studio

```
File → Open → Select android/ directory
```

Android Studio will:
1. Sync Gradle
2. Download dependencies
3. Index the project

### 3.2 Configure Local Properties

`android/local.properties` (auto-generated by Android Studio):

```properties
sdk.dir=/path/to/Android/Sdk
```

### 3.3 Configure API URL

Edit `android/app/build.gradle.kts`:

```kotlin
buildConfigField("String", "API_BASE_URL", "\"http://10.0.2.2:3000/api/v1/\"")
```

- `10.0.2.2` = localhost from Android Emulator
- For physical device: use your machine's IP (e.g., `http://192.168.1.100:3000/api/v1/`)

### 3.4 Build and Run

**From Android Studio:**
1. Select device/emulator
2. Click Run (▶)

**From command line:**

```bash
cd android

# Build debug APK
./gradlew assembleDebug

# Install on connected device
./gradlew installDebug

# Build release APK
./gradlew assembleRelease
```

### 3.5 Minimum Requirements

| Requirement | Value |
|-------------|-------|
| Android SDK | 34 (compileSdk) |
| Min SDK | 26 (Android 8.0) |
| Kotlin | 1.9.22 |
| JDK | 17 |
| Gradle | 8.2.2 |

## 4. iOS Setup (macOS only)

### 4.1 Open in Xcode

```bash
open ios/GSTBilling.xcodeproj
```

### 4.2 Configure Signing

1. Select the `GSTBilling` project in the navigator
2. Select the `GSTBilling` target
3. Go to "Signing & Capabilities"
4. Select your Apple Developer Team
5. Set Bundle Identifier: `com.wiseaccounts.app`

### 4.3 Configure API URL

Edit `ios/GSTBilling/Utilities/Constants.swift`:

```swift
enum Constants {
    #if DEBUG
    static let baseURL = "http://localhost:3000/api/v1"
    #else
    static let baseURL = "https://api.wiseaccounts.app/api/v1"
    #endif
}
```

For physical device: replace `localhost` with your machine's IP.

### 4.4 Build and Run

**From Xcode:**
1. Select simulator or device
2. Click Run (▶)

**From command line:**

```bash
# Build for simulator
xcodebuild -project ios/GSTBilling.xcodeproj \
  -scheme GSTBilling \
  -destination 'platform=iOS Simulator,name=iPhone 15' \
  build

# Run tests
xcodebuild test -project ios/GSTBilling.xcodeproj \
  -scheme GSTBilling \
  -destination 'platform=iOS Simulator,name=iPhone 15'
```

### 4.5 Minimum Requirements

| Requirement | Value |
|-------------|-------|
| Xcode | 15.0+ |
| iOS Deployment Target | 16.0 |
| Swift | 5.9+ |

## 8. Troubleshooting

| Issue | Solution |
|-------|----------|
| Prisma generate fails | Run `npm install` first, then `npx prisma generate` |
| Port 3000 in use | Change PORT in `.env` or kill the process |
| Android build fails | Run `./gradlew clean` then rebuild |
| iOS signing error | Check Apple Developer Team in Xcode signing settings |
| Database connection refused | Ensure PostgreSQL is running on localhost:5432 |
| CORS error from mobile | Update CORS_ORIGIN in `.env` or use `*` for dev |
| JWT expired immediately | Check JWT_EXPIRY value (default: 15m) |
| Room migration error | Increment version number and add migration |
